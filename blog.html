Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use methods like document.getElementById() for selecting by ID, document.querySelector() for selecting the first element that matches a CSS selector, or document .getElementsByClassName() for selecting elements by class name. For example: const element = document.querySelector('.my-class');

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to manage events triggered by its child elements. It improves performance by reducing the number of event listeners.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes in the DOM, you can use methods like setAttribute() and getAttribute(). For example, to change the src of an image: element.setAttribute('src', 'newImage.png'); To manipulate styles, you can directly modify the style property of an element. For instance, to change the background color: element.style.backgroundColor = 'blue';
